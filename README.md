# ✅ Todo App V2

A feature-rich todo application built with React and Vite.

## 🚀 Live Demo

https://todo-app-v2-pi.vercel.app

## ✨ Features

- Add, edit, and delete tasks
- Mark tasks as complete with completion timestamp
- Due date with overdue detection
- Filter tasks by All / Active / Completed
- Task counter
- Data persists with localStorage
- Soft pastel UI design

## 🛠️ Built With

- React
- Vite
- CSS

## 📚 What I Learned

- useState and useEffect hooks
- Component structure and props
- Controlled inputs in React
- localStorage with JSON.parse and JSON.stringify
- Conditional rendering with ternary operators

## 🏃 How to Run Locally

bash
git clone https://github.com/ryzaaaaamaeeeee/todo-app-v2.git
cd todo-app-v2
npm install
npm run dev
